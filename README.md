# React UI библиотеки — демо (2026)

Приложение для сравнения популярных React UI библиотек. На каждой странице показаны типичные компоненты: кнопки, формы, диалоги, табы, карточки, алерты.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173).

## Библиотеки

| Страница | Библиотека | Описание |
|----------|------------|----------|
| [/mui](/mui) | **Material UI (MUI)** | Google Material Design, богатая экосистема |
| [/antd](/antd) | **Ant Design** | Enterprise, формы и таблицы |
| [/chakra](/chakra) | **Chakra UI** | Удобный DX, доступность, тёмная тема |
| [/mantine](/mantine) | **Mantine** | 100+ компонентов, хуки, формы |
| [/radix](/radix) | **Radix UI** | Headless примитивы, полный контроль над стилями |

## Компоненты на страницах

- **Кнопки** — варианты (primary, outline, ghost и т.д.)
- **Формы** — Input, Select, Checkbox, Switch
- **Диалог / Модалка**
- **Табы**
- **Карточка**
- **Alert / уведомления**
- **Chip / Tag / Badge** (где есть)
- **Dropdown** (Radix)
- **Slider, Tooltip** (Radix)

## Стек

- React 19 + TypeScript
- Vite 7
- React Router 7
